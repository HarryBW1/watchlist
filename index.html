One last thing — connect TMDB

Watchlist uses The Movie Database to fetch posters, descriptions, ratings, and streaming provider info. A TMDB account and API key are free and take about 2 minutes to set up.

Signed in as
1 Go to themoviedb.org/signup and create a free account
2 Open Settings → API, click Create, choose Developer
3 Copy your API Key (v3 auth) and paste it below

Your key is saved to your account — you'll never need to enter it again on any device.
